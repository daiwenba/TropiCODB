The TropicoDB aims to be the leading platform providing omics data and tools for researchers, as well as breeding data and tools for breeders. We offered a series online bioinformatics tools, hoping to deal with some key issues in the whole industry chain in tropical crops. In the near future, we will continue to update this database as long as we can.The only comprehensive database focusing on tropical crops. We collected 8 species up to 2024.

July 25, 2023: TropiCODB database entered production.

August 25, 2023: Blast tool built.

September 25, 2023: Jbrowse tool built.

December 20, 2023: Search tool launched.

December 20, 2024: Genome module completed.

July 10, 2024: New transcriptome module added, including eFP-gene and heatmap functions.

August 1, 2024: Ability to display population data added.

September 10, 2024: Ability to display metabolic data added.

October 21, 2024: Phenotypic data added.

December 21, 2024: TropiCODB database version 1.0 officially released.
